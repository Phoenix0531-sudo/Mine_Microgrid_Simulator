# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-06-08

### Added
- Initial public release
- 8760-hour hourly energy balance simulation engine
- Solar PV and wind turbine modeling via pvlib and windpowerlib
- Real-time weather data from Open-Meteo API
- Interactive Plotly dashboards (3D energy flow, heatmap, radar, waterfall)
- Multi-dimensional KPIs (renewable penetration, self-consumption, grid independence)
- Economic analysis (NPV, IRR, LCOE, payback period)
- Sensitivity analysis and optimization recommendations
- Async computation with progress display
- Memory optimization for large datasets
- Performance monitoring system
- Apple-style modern UI with dark gradient theme
