## Get started

### Install

```bash
git clone https://github.com/Phoenix0531-sudo/Mine_Microgrid_Simulator.git
cd Mine_Microgrid_Simulator
pip install -r requirements.txt
```

### Usage

```bash
streamlit run app.py   # or entry documented in modules/
pytest tests/
```
